# Digital-Museum
The Digital Museum is an innovative web project designed to transform a traditional website into an immersive, interactive experience. Built entirely with HTML,JAVA and CSS, this project reimagines how information can be presented by simulating the look and feel of a museum. Instead of static pages, the site is structured as a series of “gallery rooms”.
Each gallery showcases “exhibits” using CSS grids and flexbox layouts. Exhibit cards display images, titles, and short descriptions, with hover effects that reveal additional details or animations. This design not only makes the content visually appealing but also demonstrates mastery of responsive layouts and interactive styling. To enhance the academic feel, the museum includes a timeline wall, where historical events or milestones are arranged horizontally with smooth scrolling and transitions. Another highlight is the interactive map, a floor plan of the museum created with HTML image maps and styled hover effects, allowing users to click and explore different galleries intuitively.
